Q : Apa itu website AsikAja?
AsikAja adalah website non-resmi/non-official dari Youtube, website ini adalah web pribadi Developer dengan memanfaatkan API dari Youtube. Pengembangan website ini bermula dari masalah pribadi Developer yang mungkin dialami juga oleh banyak orang, yaitu terdistraksi oleh video rekomendasi youtube saat hanya berniat menonton konten atau informasi tertentu, seringnya muncul video yang kurang berfaedah, dan larutnya Developer untuk terus menonton hingga berjam-jam hingga sulit berhenti.
Q : Di youtube ada video baru di channel, kenapa di sini tidak ada?
Youtube Lite menggunakan API Youtube untuk menampilkan video. Beberapa video, short, atau livestreaming mungkin tidak muncul dikarenakan tidak diberikan akses oleh youtube melalui API. Namun, bisa jadi karena mekanisme refresh otomatis video pada channel yang baru akan aktif jika waktu penambahan video dari API ke database dengan waktu saat ini belum lewat dari 12 jam.
Q : Mengapa saat saya tekan tombol next tidak terjadi apa-apa?
Ada dua kemungkinan. Pertama, batas kuota dari API Youtube telah habis, jadi baru bisa direfresh keesokan harinya dikarenakan satu kuota API dipakai bersama untuk menampilkan konten yang selanjutnya konten tersebut dapat ditampilkan juga untuk user lain. Kedua, terjadi pembatasan atas video yang ditampilkan oleh youtube, sehingga channel tertentu tidak dapat menampilkan video terdahulu melalui API. 8/9/2025
